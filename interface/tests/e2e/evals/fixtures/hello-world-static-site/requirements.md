# Hello World Benchmark

Build a tiny static web page for Aura's benchmark suite.

Requirements:

- Create `index.html`.
- Set the page title to `Hello Aura`.
- Render a heading `Hello from Aura`.
- Render a paragraph with `data-testid="tagline"` and the text `This page was generated by Aura.`.
- Keep the project dependency-free and make sure both `npm run build` and `npm run test` pass.
