# iOS App Store metadata scaffold

Populate this directory before a real App Store / TestFlight submission.

Recommended structure:

- `en-US/name.txt`
- `en-US/subtitle.txt`
- `en-US/description.txt`
- `en-US/keywords.txt`
- `en-US/release_notes.txt`

Also prepare:

- `../screenshots/` for App Store screenshots
- App Review notes / demo credentials in App Store Connect
- Privacy policy URL
- Support URL
- Age rating and App Privacy answers

Do not upload placeholder copy to production.
