# Android Play metadata scaffold

Populate this directory before a real Google Play submission.

Recommended structure:

- `android/en-US/title.txt`
- `android/en-US/short_description.txt`
- `android/en-US/full_description.txt`

Also prepare in Play Console:

- high-resolution icon
- phone screenshots
- tablet screenshots if you plan to support tablets in the listing
- privacy policy URL
- Data safety answers
- content rating
- app access / review instructions

Do not upload placeholder copy to production.
